‚ÄúDSL KeyPad‚Äù is a¬†tool written on¬†[AutoHotkey 2.0](https://www.autohotkey.com/), designed for inputting a wide range of characters using hotkeys, [Compose](https://en.wikipedia.org/wiki/Compose_key)-like mode and auxiliary functions. Its primary focus is on enhancing input capabilities for Latin and Cyrillic scripts, allowing typing in multiple languages without the need for separate keyboard layouts for each language.

You can check the docs for this tool [here](https://demernkardaz.github.io/DSL-KeyPad/). Downloading available on [Releases](https://github.com/DemerNkardaz/DSL-KeyPad/releases) & [SourceForge](https://sourceforge.net/projects/dsl-keypad/files/).

### Shortlist of Features

- **Multilayer Hotkeys**: Including different for English (‚ÄúLatin set‚Äù) and Russian (‚ÄúCyrillic set‚Äù).  
  E.g., \[en.\] <kbd>RAlt + A or Z</kbd> ‚Üí `ƒÇ`/`≈ª`, then \[ru.\] <kbd>RAlt + –§ or –Ø</kbd> ‚Üí `—≤`/`—¶`. Supports user-defined key bindings.

- **Compositing Mode, or ‚ÄúForge‚Äù**: Converts a sequence (‚Äúrecipes‚Äù) of one set of characters into another.  
  E.g., `AE` `OE` `TH` `–Ü—¶` `–Ø–ï` `‚∞¶‚∞§` `Ups` ‚Üí `√Ü` `≈í` `√û` `—®` `‘ò` `‚∞©` `∆±`. Supports user-defined recipes.

- **Alternative Input**: A set of modes for various non-Latin/Cyrillic scripts (primarily historic), IPA, and mathematical symbols.  
  E.g., \[Runic `·õ¢·öπ·õñ·ö±·õè·ö§·ö¢·õÅ·õü·õà·ö®·õä·õû·ö†·ö∑·ö∫·õÉ·ö≤·õö·õâ·ö≥·ö°·õí·öæ·õó·õ¨`\] \[Glagolitic `‚∞â‚∞ú‚∞ñ‚∞ç‚∞Ö‚∞ê‚∞É‚∞û‚∞õ‚∞à‚∞ò‚∞ü‚∞ó‚∞Ç‚∞Ä‚∞í‚∞ì‚∞°‚∞ù‚∞î‚∞è‚∞ã‚∞†‚∞Å‚∞£‚∞¶`\].

- **Glyph Variations**: Allows entering alternative variants of symbols, if available.  
  E.g., `A` ‚Üí `·¥¨` `ùê¥` `ùêÄ` `ùë®` `ùô∞` `Ôº°` `·¥Ä` `ùîÑ` `ùï¨` `ùíú` `ùìê` `ùî∏`.

- **Switch Between Entering Unicode Symbols/HTML Code/ $\LaTeX$ Codes**, if available.  
  E.g., `ƒÇ` `«¢` ‚Üí `&Abreve;` `&#482;` ${\color{darkorange}\texttt{and}}$ `\u{A}` `\={\AE}` ${\color{darkorange}\texttt{or}}$ `\breve{A}` `\bar{\AE}` $\breve{A}$ $\text{«¢}$.

- **Search Symbols in Local Library by Tags**: Allows searching for symbols in the local library by tags for easy input.  
  E.g., `latin small ligature turned oe with short stroke` or shortened `oe tur str` ‚Üí `Í≠Ç`.

- **Internal Keyboard Layouts**: Includes support for user-defined layouts.

**Strong 2** recommendation: turn off all AutoHotkey processes while playing games with sensitive anti-cheat systems.
